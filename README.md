# PyServerlessDB

A Serverless DB for hobby python projects

# Installation
- clone/download repo
    ```
    git clone https://github.com/dmdhrumilmistry/pyserverlessdb.git
    ```
- change directory
    ```
    cd pyserverlessdb
    ```
- run tests
    ```
    python3 setup.py test
    ```
- if test is ok, then install
    ```
    pip install -e .
    ```
